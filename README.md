# Room Booking (Kokoushuonevaraaja)

## Alkuun
### Kloonaa repository

#### Avaa terminaali ja siirry kansioon, johon haluat kloonata repositoryn

```bash
cd valitsemasi_kansio

git clone https://github.com/Toni-Andstrom/Room_Booking.git

cd Room_Booking

```
## Vaatimukset

- Node.js v24.13.0 tai uudempi (LTS-suositus)

### Seuraavat komennot ajetaan projektin juurihakemistosta:

## Backend asetukset

```bash
cd backend
node server.js
```

## Frontend asetukset

```bash
cd frontend
npm install
npm run dev
```
